Setup Help ‚Äî Local Python Bridge

If Pyodide fails in your browser you can run a local Python bridge to convert WAV ‚Üí .rshw.

Steps (Windows PowerShell):

python -m venv .venv
source .venv/Scripts/activate
pip install -r tools/requirements.txt
python tools/py_bridge_server.py

Steps (macOS / Linux):

python3 -m venv .venv
source .venv/bin/activate
pip install -r tools/requirements.txt
python tools/py_bridge_server.py

Optional: for best compatibility, install .NET SDK so the returned .rshw uses the exact BinaryFormatter serialization: 

choco install dotnet-sdk -y  # Windows (chocolatey)
brew install --cask dotnet-sdk # macOS (Homebrew)
sudo apt install dotnet-sdk-7.0   # Debian/Ubuntu (apt)
dotnet --version

After starting the bridge, click Check Bridge and then Convert to SPTE rshw.

Download & Run Helper

The simplest, fastest helper is a small Python script you can run locally ‚Äî no .NET required. It starts a local HTTP server that the web UI will talk to for WAV ‚Üí .rshw conversion.

Option A ‚Äî Python helper (recommended)

Download the Python helper script
If you don't have Python installed, get it from python.org. Double-clicking the .py file will not run it ‚Äî use the terminal or the included runner scripts.

Quick run (Windows):

python -m venv .venv
.venv\Scripts\activate
pip install -r tools/requirements.txt
python tools/py_bridge_server.py

Quick run (macOS / Linux):

python3 -m venv .venv
source .venv/bin/activate
pip install -r tools/requirements.txt
python tools/py_bridge_server.py

The server listens on http://127.0.0.1:5000. After you start it, click Check Bridge and then Convert to SPTE rshw.

Option B ‚Äî .NET helper (optional)

Use this only if you want the BinaryFormatter-serialized .rshw.

dotnet publish tools/rshw_serializer -c Release -r win-x64 /p:PublishSingleFile=true -o dist

Double-click the produced executable in dist\ to run it. It will listen on http://127.0.0.1:5000.

After starting the helper, click I've started the helper below and the app will resume the pending conversion (if any).
